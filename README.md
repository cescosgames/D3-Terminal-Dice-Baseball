> **Terminal dice baseball for the Raspberry Pi Zero.**
> Just you, the CPU, and 9 D3's

---

Wanted to load a simple baseball sim on my raspberry pi zero so here we are.

Load onto pi with your preffered method
```bash
scp baseball_dice.py pi@<ip>:~/
ssh pi@<ip> python3 baseball_dice.py
```

then run and play!

```bash
python3 baseball_dice.py
```
